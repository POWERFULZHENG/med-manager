这份文档旨在为**家庭常用药管理系统**的前端开发提供视觉与交互的指导标准。该方案严格遵循 Vue3 + Element Plus 的技术栈要求，并基于“模块化”与“最小可用版本（MVP）”原则进行设计。

------

## 1. 视觉风格与全局规范

### 1.1 视觉调性

- **主色调**：医疗蓝 (`#409EFF`) 或 环保绿 (`#67C23A`)，营造专业、洁净且充满生机的视觉感受。
- **状态色**：
  - **正常 (NORMAL)**：绿色 (`#67C23A`)。
  - **临期 (EXPIRING)**：黄色/橙色 (`#E6A23C`)。
  - **过期 (EXPIRED)**：红色 (`#F56C6C`)。
- **字体**：优先使用系统默认无衬线字体，正文不小于 **14px** 以确保可读性。

### 1.2 全局布局 (Layout)

- **侧边栏 (Sidebar)**：深色或白色背景，收纳导航菜单（首页、药品管理、库存管理、出入库记录、OCR识别、用户管理）。
- **顶栏 (Header)**：展示当前页面面包屑、用户头像、消息通知（过期预警）及退出登录。
- **主体区 (Main)**：灰色背景 (`#F0F2F5`)，内容承载于白色卡片中。

------

## 2. 核心页面设计

### 2.1 仪表盘 (Dashboard)

- **顶部统计卡片**：使用 `el-statistic` 展现总药品数、总库存、今日出入库次数，特别突出**临期**与**过期**数量（红色高亮）。
- **预警列表**：下方左侧展示“最近 5 条过期预警”，点击可直接跳转库存详情。
- **快捷入口**：右侧提供“扫码/拍照入库”的大按钮，方便用户快速操作。

### 2.2 药品管理 (Medicine Management)

- **查询区**：支持药品名称、厂家、剂型的输入框及重置按钮。
- **列表区**：使用 `el-table` 展示药品基础信息（规格、单位、厂家），操作列包含“编辑”、“查看库存批次”和“删除”。
- **弹窗设计**：新增/修改药品时采用 `el-dialog`，表单需对必填项（名称、规格）进行校验。

### 2.3 库存管理 (Stock Management)

- **批次视图**：核心展示“批次号 + 数量 + 失效日期 + 存放位置”。
- **状态列**：根据 `expire_date` 自动计算并显示状态标签（正常/临期/过期）。
- **入库操作**：支持选择现有药品或新建药品，必须输入批次号和失效日期。
- **出库操作**：点击列表中的“出库”，弹出简易对话框输入数量及备注，需校验出库数是否大于当前批次余量。

### 2.4 OCR 识别页面 (OCR Processing)

- **上传区**：醒目的 `el-upload` 拖拽上传或点击上传图片。
- **对比核对区**：
  - **左侧**：展示上传的原始图片。
  - **右侧**：展示识别后的表单（药品名、失效日期、数量等），所有字段均可人工二次修改。
- **提交逻辑**：点击“确认识别并入库”，将数据提交至后端库存接口。

------

## 3. 关键交互规范 (UX)

### 3.1 响应式适配

- **移动端切换**：当屏幕宽度小于 **768px** 时，侧边栏自动隐藏并转换为汉堡菜单；表格转为列表卡片模式展示。

### 3.2 异常与反馈

- **操作反馈**：所有增删改查操作成功后，需弹出 `el-message` 提示。
- **删除二次确认**：删除操作需触发 `el-message-box` 二次确认，防止误删。
- **加载状态**：查询列表和 OCR 识别期间，需在对应组件上显示 `v-loading`。

------

## 4. UI 组件映射表

| **功能点**        | **Element Plus 组件** | **说明**                         |
| ----------------- | --------------------- | -------------------------------- |
| **主导航**        | `el-menu`             | 垂直布局，支持折叠               |
| **数据列表**      | `el-table`            | 带分页，支持表头排序             |
| **入库/出库弹窗** | `el-dialog`           | 保持表单简洁，减少层级           |
| **日期选择**      | `el-date-picker`      | 仅允许选择当天及之后日期（入库） |
| **预警标签**      | `el-tag`              | 采用不同主题色区分库存状态       |
| **数据分页**      | `el-pagination`       | 统一放在列表底部右侧             |
